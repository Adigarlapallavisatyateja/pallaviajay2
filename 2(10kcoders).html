ELEMENTS

The combination of opening,closing tag and with content between the tags is called elements.

Element can have attributes also.

TYPES OF ELEMENTS:

1.Block level element:

The tag which occupies whole width and creates a new line.

EX:h1-h6,p,div,semantic elements.

2.Inline-block elements:

The elements which occupies only required space and does not create a new line and we can apply width,padding,margin,height and padding properties.

EX:img,iframe,input,button

3.Inline element:

The elements which occupies only required space and doesnot create a new line.

EX:span,p,strong,i
